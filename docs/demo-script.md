# Three Minute Demo Script

## 0:00-0:25 Problem

Small builder teams miss cash hackathons because deadlines, eligibility, required assets, and public-action risks are scattered across platforms.

## 0:25-1:20 Scan

In Slack, ask:

```text
Find cash-backed online hackathons ending in the next 14 days.
```

Show BountyOps returning:

- Slack Agent Builder Challenge as do-first
- Qwen Cloud as do-second
- Reddit game challenge as backup
- Kaya as skip due team and Bangalore requirements

## 1:20-2:05 Drill-down

Ask:

```text
Create a submission checklist for the Slack Agent Builder Challenge.
```

Show checklist:

- public demo video under 3 minutes
- architecture diagram
- Slack developer sandbox URL
- judge access for required emails
- text description
- no public submission until approval

## 2:05-2:40 Slack-native workflow

Show the App Home and suggested prompts. Mention that Slack MCP can connect workspace context, drafts, and canvases while preserving human approval boundaries.

## 2:40-3:00 Close

State the product value: BountyOps converts hackathon discovery into a ranked, executable Slack workflow for teams who need to ship before the deadline.
